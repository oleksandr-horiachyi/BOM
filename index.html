How it works
Four steps, two files, done.
No training required. If you can open Excel, you can use FabTools.
STEP 01
📂
Upload your files
Select your BOM file and your LOG file — both are standard Excel files (.xlsx or .xlsm) that you already use in your workflow.
💡 BOM = your Bill of Materials spreadsheet.
LOG = your production tracking sheet with part codes.
→
STEP 02
☑️
Choose LOG sheets
Your LOG file may contain multiple sheets — one per production unit (e.g. U-1000, U-2000). Select only the units you want to schedule.
💡 You'll see the sheet names from your file. Tick the ones you need — you can select all or just one.
→
STEP 03
⚡
Run
Click Run. The tool matches part codes from your BOM against your LOG, calculates quantities, and builds your reports automatically.
💡 Takes a few seconds. Any mismatches or missing codes are flagged in a separate Checks sheet.
→
STEP 04
📥
Download reports
Download your Combined BOM and Production Schedule as Excel files. Open them directly in Excel — no special software needed.
💡 Files are formatted and ready to use: headers styled, columns auto-fitted, filters applied.

FAQ
Questions you might have.
🔒 Are my files safe?
Yes. Your files are processed directly in your browser (or on a temporary server session) and are never stored, logged, or shared. Once you close the tab, everything is gone. We never see your data.
📄 What is a LOG file?
The LOG file is a standard Excel spreadsheet (.xlsx) — your production tracking sheet that contains part codes organized by unit. Each sheet tab in the file typically represents one production unit (e.g. U-1000). It is not a system log or text file.
📊 What format must my files be in?
Both the BOM and LOG files must be Excel format (.xlsx or .xlsm). The tool expects part codes in column A, starting from row 3. Download the sample files above to see the exact expected layout.
⚙️ Do I need to install anything?
Nothing at all. FabTools runs entirely in your web browser. No software installation, no account creation, no license required. Just open the tool and upload your files.
🔢 What if some part codes don't match?
The tool automatically generates a Checks sheet and a Missing parts sheet in your output files, listing every code that exists in your BOM but wasn't found in the LOG, and vice versa. Nothing gets silently dropped.
💸 Is it free?
Yes, completely free. FabTools is a personal project. All tools are free to use with no usage limits, no subscription, and no hidden costs.
